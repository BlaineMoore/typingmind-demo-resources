# TypingMind Public Domain Analysis Agent Files

The TypingMind "agents" are AI Personas that let you replace the global system message with one specific to your needs, along with further customizing the agent with access to specific Knowledge Base tags, LLM models, and discussion starter prompts.

The template used for the system model can be read in [the system-template markdown file](/agents/system-template.md), and each agent has a customized version based on how they would be used in the demo or how somebody might want to experiment on their own after the demo.

To import each of these avatars:

![TypingMind screenshot with arrows pointing at where to click to import an agent](./TypingMind_-_Importing_Agents.jpg)

1. Click the **Agents** button in the side navigation.

2. Click the *expand* arrow next to the **Create AI agent** button.

3. Click the **Import from JSON** button.

4. Choose your downloaded agent JSON file.

5. *(Optional):* Copy the Avatar link if you don't like the randomly generated one.

6. *(Optional):* Choose an appropriate **tag** to limit your Knowledge Base searches if necessary.
