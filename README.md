# TypingMind Demo Resources

These resources were used to put together our "[How to use TypingMind](https://apexauthors.com/training-636-typingmind/)" at Apex Authors on May 26, 2026.

You are welcome to copy and use any of the resources listed here.

Special thanks to [Project Gutenberg](https://gutenberg.org) for providing public domain materials that could be used as examples for this demonstration. All books provided comply with [the Project Gutenberg license](/public-domain-books/project-gutenberg-license.md), and you can download both [the original ePub files](/public-domain-books/epubs/) and [our cleaned markdown files](/public-domain-books/markdown/) from the [/public-domain-books/ folder](/public-domain-books/).

The markdown files have removed the front and back matter added by Project Gutenberg to avoid confusing the LLMs with modern language that is not directly relevant to the novels, and in some cases have had their chapter designations cleaned up and made more consistent. See the attached [gutenberg-cleaner skill](/gutenberg-cleaner/) for converting text versions of the novels into Markdown. (*Note: we used [Calibre ebook management](https://calibre-ebook.com/) to convert from ePub to TXT with markdown output for the initial conversions, and have not tested direct TXT conversions with the skill though it is setup to potentially handle that.*)
